<h2>How To Build</h2>
<h3>Building with cmake and make</h3>
Navigate to RetroSpaceGame/ </br>
cmake -B build</br>
cd build</br>
make</br>

<h2>How To Play</h2>
A - Move player left</br>
D - Move player right </br>
Space - Shoot</br>
R - Reset Player</br>
